# Проверка доступных моделей в LLMost

## Проблема

Модель не найдена в LLMost. Нужно проверить точные названия моделей.

## Решение

1. **Откройте сайт LLMost:** https://llmost.ru/
2. **Перейдите в раздел "Модели"** (в меню наверху)
3. **Найдите точные названия моделей** - они могут отличаться от стандартных OpenAI
4. **Используйте точное название** в `.env`

## Примеры возможных названий

- `gpt-3.5-turbo-0613`
- `gpt-3.5-turbo-1106`
- `gpt-4-0613`
- `gpt-4-turbo`
- Или другие варианты, указанные на сайте

## Обновление .env

После того, как найдете правильное название модели, обновите `.env`:

```env
RAGAS_LLM_MODEL=точное_название_модели_из_llmost
RAGAS_OPENAI_BASE_URL=https://llmost.ru/api/v1
RAGAS_OPENAI_API_KEY=your_llmost_api_key_here
```

## Альтернатива

Если не можете найти правильное название модели, используйте Ollama локально:

```env
RAGAS_LLM_MODEL=llama3.1:8b
RAGAS_OPENAI_BASE_URL=http://localhost:11434/v1
RAGAS_OPENAI_API_KEY=ollama
```
