# Устранение проблем с LLMost API

## Проблема: APIConnectionError при подключении к LLMost

### Возможные причины:

1. **Неправильный URL API**
   - Проверьте документацию LLMost: https://llmost.ru/docs
   - Возможно, URL должен быть другим (например, без `/v1` или с другим путем)

2. **Неправильный формат API ключа**
   - Убедитесь, что ключ скопирован полностью
   - Проверьте, что нет лишних пробелов

3. **Блокировка брандмауэром/антивирусом**
   - Временно отключите брандмауэр/антивирус для проверки
   - Добавьте исключение для Python/uv

4. **Недостаточно баланса**
   - Проверьте баланс на сайте LLMost
   - Убедитесь, что есть средства для запросов

### Решение: Использовать Ollama локально для RAGAS

Если LLMost не работает, используйте Ollama локально:

```env
# Основной бот - Groq (оставляем как есть)
OPENAI_BASE_URL=https://api.groq.com/openai/v1
LLM_MODEL=llama-3.1-8b-versatile
OPENAI_API_KEY=your_groq_key

# RAGAS - Ollama (локально, без проблем с подключением)
RAGAS_LLM_MODEL=llama3.1:8b
RAGAS_OPENAI_BASE_URL=http://localhost:11434/v1
RAGAS_OPENAI_API_KEY=ollama
```

### Проверка подключения к LLMost

1. Откройте документацию: https://llmost.ru/docs
2. Проверьте правильный URL API
3. Проверьте формат API ключа
4. Попробуйте сделать тестовый запрос через curl или Postman

### Альтернативы

Если LLMost не работает:
- **Ollama** - локально, без лимитов
- **GigaChat** - российский, работает без VPN
- **Yandex GPT** - российский, работает без VPN
