
## Файл `docs/idea.md`

### 1. Название проекта
```
# BankAssist — AI-driven ассистент для банков
```

### 2. Описание проекта
```
БанкАссист — это инновационная система помощи клиентам банков, основанная на технологиях искусственного интеллекта и обработки естественного языка. Проект направлен на повышение качества обслуживания клиентов, ускорение процесса решения повседневных вопросов и снижение нагрузки на сотрудников банка. БанкАссист способен автоматически обрабатывать обращения клиентов, давать консультации по продуктам и услугам банка, анализировать финансовую ситуацию клиента и предлагать персонализированные советы. Система поддерживает взаимодействие с пользователями через текст и голос, способна распознавать и интерпретировать изображения документов.
```

### 3. Целевая аудитория
```
Проект ориентирован на две ключевые группы пользователей:
- Клиенты банка, нуждающиеся в поддержке при обращении за услугами (оформление кредита, открытие вклада, получение карты, консультация по тарифам и условиям продуктов).
- Сотрудники банка, использующие БанАссист для автоматизации рутинных операций и улучшения собственной производительности труда.
```

### 4. Основные функции и возможности
```
Основные функции Банкассиста включают:
- Ответы на частые вопросы клиентов по услугам и продуктам банка.
- Оформление заявлений на кредиты, вклады и дебетовые карты.
- Автоматическое заполнение анкет и документов на основании предоставляемых клиентом сведений.
- Предоставление рекомендаций клиенту по управлению финансами и инвестициями.
- Возможность отправки сканов документов и фотографий, а также их автоматической интерпретации системой.
- Подключение к внутренним системам банка для быстрого предоставления актуальной информации о состоянии счетов, транзакциях и лимитах.
```

### 5. Используемые технологии и подходы из курса
```
Для реализации Банкассиста планируется использование следующих технологий и подходов, изучаемых в рамках курса:
- **LLM через API**: Интерфейсы взаимодействия с моделями OpenAI позволят обеспечить высокое качество ответов и понимания сложных запросов клиентов.
- **Structured Output**: Механизмы структурированного вывода необходимы для четкого представления решений и аналитики системы.
- **Мультимодальность**: Реализация работы с текстом, голосом и изображениями позволит пользователям удобно взаимодействовать с ассистентом разными способами.
- **RAG (Retrieval-Augmented Generation)**: Эта технология обеспечит интеграцию с внутренней базой знаний банка, позволяя быстро находить нужную информацию.
- **Agent Patterns & Workflows**: Использование архитектуры агентов и сложных графов (LangGraph) повысит эффективность выполнения сложных многошаговых действий.
- **MCP (Model Context Protocol)**: Для расширения функционала ассистента за счет сторонних инструментов и приложений.
```

### 6. План развития проекта по темам курса
```
| Неделя | Тема изучения     | Действия над проектом |
|--------|-------------------|----------------------|
| 1      | Установка среды   | Настройка базовой инфраструктуры и прототипирование минимального рабочего прототипа. |
| 2      | Простые диалоги   | Разработка простого чатбота с ограниченным набором функций и фреймворком для дальнейшего масштабирования. |
| 3      | RAG               | Создание механизма интеграции с внешней базой знаний банка для расширенного информирования клиентов. |
| 4      | Multimodality     | Добавление модулей для обработки голосовых запросов и изображений (например, скан паспорта). |
| 5      | Agents            | Организация сложного поведения помощника путем построения графа задач и пошаговых алгоритмов. |
| 6      | Final Presentation| Завершение разработки и подготовка итоговой презентации перед преподавателем и одногруппниками. |
